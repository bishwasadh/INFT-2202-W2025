Project Overview

This assignment focuses on creating a functional blog page that retrieves data from external APIs. The main features include:

  • Fetching blog posts from JSONPlaceholder API
  • Displaying posts with relevant images
  • Implementing responsive design using Bootstrap
  • Creating reusable components with JavaScript

  • Modular JavaScript organization
