This assignment demonstrates the use of AJAX and Fetch API calls to create a dynamic blog page.

Blog Feature

Our blog page demonstrates fetching data from external APIs:

  • Retrieves blog posts from JSONPlaceholder API
  • Displays images relevant to each post
  • Uses Bootstrap card layout with Flexbox

Technologies Used

This project showcases several modern web technologies:

  • Fetch API for asynchronous HTTP requests
  • Bootstrap 5 for responsive design
  • Flexbox for flexible layouts
  • Dynamic content generation with JavaScript
